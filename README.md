## macOS Compatibility
All "final release" versions of macOS from 10.6-10.13 are supported. This includes:
- 10.6.8 (10K1136)
- 10.7.5 (11G63)
- 10.8.5 (12F2560)
- 10.9.5 (13F1911)
- 10.10.5 (14F2511)
- 10.11.6 (15G22010)
- 10.12.6 (16G2136)
- 10.13.6 (17G14042)

Newer releases may function by using OpenCore Legacy Patcher.
